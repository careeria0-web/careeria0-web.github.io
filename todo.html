Tehtävälista-sovellus

Roolini: Kehittäjä

Tekniikat: HTML5, CSS3, JavaScript (Vanilla JS), LocalStorage API

Projektin idea: Kehittää yksinkertainen tehtävälista-sovellus, jossa käyttäjät voivat lisätä, merkitä valmiiksi ja poistaa tehtäviä. Projekti auttoi oppimaan JavaScript:n perusteita kuten DOM-manipulointi, tapahtumankäsittely ja LocalStorage API:n käyttö.

Ominaisuudet:

  • Tehtävien lisääminen syöttökenttään
  • Tehtävien merkitseminen valmiiksi (rasti-painikkeella)
  • Tehtävien poistaminen
  • Tehtävien pysyvä tallennus LocalStorage API:lla
  • Tilastot: yhteensä tehtävät ja valmiiksi merkityt tehtävät
  • Responsiivinen design
